# Code of Conduct

Thank you for being part of Retreever.  
This project follows a simple rule: **be respectful, be constructive, and help the community grow**.

## Our Standards

### We encourage:
- Open, friendly, and constructive discussions  
- Clear communication and collaborative problem-solving  
- Respect toward contributors of all skill levels  
- Helping others understand the project and its codebase

### We do not accept:
- Harassment, insults, personal attacks, or discrimination  
- Dismissive or aggressive behavior  
- Spam, self-promotion, or irrelevant content  
- Any conduct that disrupts the project or its community

## Reporting Issues

If someone behaves inappropriately or violates this code, please report it privately to:

**rajugowda.towork@gmail.com**

All reports will be reviewed. Actions may range from a warning to removal from project participation.

## Responsibilities of Maintainers

Project maintainers:
- Clarify guidelines and expectations  
- Review contributions fairly  
- Take corrective action when a violation occurs  
- Have the right to remove comments, issues, PRs, or contributors that violate this code  

## Scope

This Code of Conduct applies:
- In GitHub issues, pull requests, and discussions  
- In any communication related to the project  
- In any community space where Retreever is represented

## License

This Code of Conduct is adapted from the Contributor Covenant (v2.1), with simplifications tailored for this project.

By contributing to Retreever, you agree to follow this Code of Conduct.
